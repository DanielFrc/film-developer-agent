**Date:** 2026-06-23  
**Tag:** `v0.2.0`

## Summary

This release turns the app from a portfolio demo into a **daily darkroom workflow**: chart lookups, sink-ready **session cards** (no LLM required), a personal **library**, and per-combination **workbook notes** — all client-side except gold chart data from your local API.

---

## Highlights

### Session card (default workflow)

After confirming a lookup on **Search**, get a printable checklist:

- Chart time vs your working time
- Developer + water volumes from dilution and tank size (Preferences)
- Stop bath, agitation, pre-soak
- **Save session** to Library without generating a full LLM recipe

### Personal knowledge layer

- **Library** — saved sessions, recipes, favorites, film notes
- **Workbook** — adjusted scan time, pre-soak override, rolls developed (+1), outcome density/grain, free-text notes
- **Film enrichment** — box speed / typical EI pre-fills Search
- **Regenerate with notes** — one-shot LLM call merging workbook + preferences

### Compare & polish

- **`/compare`** — same film / format / ISO across developers (gold chart only)
- **PWA** — install on phone; saved recipes readable offline
- **Print** — session card and recipe summary layouts
- **`GET /compare`** API endpoint

### Navigation

Workflow: **Search · Compare · Library** | Data: **Preferences · Explorer · Dashboard**

---

## Upgrade notes

- **Library backup:** export v3 includes darkroom prefs and workbook fields. v1 and v2 imports still work.
- **Preferred developers:** now stored as comma-separated text (`rodinal, d-76, xtol`). Old array backups migrate on import.
- **Gold data:** unchanged — run `film-agent pipeline` locally; `data/` remains gitignored.

## Quick start (existing users)

```bash
git pull
pip install -e ".[dev]"
cd apps/web && npm install && cd ../..
make check
./scripts/dev.sh
```

Import your library JSON from Preferences if switching browsers or devices.
